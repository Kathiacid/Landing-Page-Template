<h1>Landing page template</h1>
<p>Para este proyecto he utilizado las siguientes tecnologías:</p>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
<p>Este proyecto me tomó aproximadamente 10 horas al no tener una idea clara de lo que quería. Además, lo que se me ocurría lo investigué hasta llegar al resultado final, pues no todo es algo que sepa de memoria. Está codificado, por obvias razones, en inglés y de igual forma quise que el contenido también lo estuviera.</p>

<h2>Tecnologías</h2>
<ul>
  <li><b>HTML:</b> <p>Estructura básica del proyecto, no tiene gran cosa, solo contenedores dentro de contenedores con más contenedores.</p></li>
  <li><b>CSS:</b> <p>Implementé el uso de transiciones, hovers, menú hamburguesa (sin Bootstrap), media query y WebKit para aplicar estilos específicos del navegador.</p></li>
  <li><b>JavaScript:</b> <p>Implementé un carrusel de imágenes en el header, para el cual tuve que implementar el JS de <b>Swiper</b>, el cual fui aprendiendo a ocupar a medida que investigaba sobre cómo implementarlo.</p></li>
</ul>
